* SQL College Project
A graphical SQL interface for managing professors and students of an institution.

** Building
This project makes use of the GNU build system (autotools). Standard pipeline applies here as well.

*** Dependencies
- *GTK4*
- *MariaDB* /or/ *MySQL*
- *texlive*

*** Compiling
#+BEGIN_SRC sh
autoreconf -fi
./configure
make DBCLIENT_USR="<your SQL user>" DBCLIENT_PWD="<your SQL user password>"
#+END_SRC
/Compiling and running with SQL user as "root" not tested and not encouraged!/

*** Running
#+BEGIN_SRC sh
./src/manager <user email> <user password>
#+END_SRC

*** Issues

**** Access Denied:
Make sure your database user has enough permissions to do anything with the database.
